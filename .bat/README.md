## ⚠️ 주의사항 

- ### 윈도우 환경에서만 동작합니다.

- ### "알 수 없는 게시자" 관련 팝업창은 무시하고 진행하셔도 무방합니다.

  해당 경고는 `run.bat` 파일이 구동에 필요한 ffmpeg 및 필수 패키지를 다운로드하는 과정에서 시스템이 안내하는 정상적인 현상입니다.

  혹시 그래도 불안하시다면, `run.bat` 파일을 ChatGPT나 Gemini 같은 AI에게 보여주고 검증해 달라고 요청해 보시면 안전한 코드인지 확인해 줄 것입니다.

- ### 화면이 멈춘 것처럼 보이더라도 에러 메시지가 뜨지 않는다면 정상적으로 실행 중인 것입니다.
  
  별도의 진행률 바(Progress Bar)가 표시되지 않으므로, 프로세스가 완료될 때까지 강제로 창을 종료하지 마시고 여유 있게 대기해 주시기 바랍니다.

## 🚀 NVIDIA GPU (CUDA) 하드웨어 가속 안내

NVIDIA 그래픽 카드를 사용하시는 경우 아래 사항을 확인해 주세요.

### 그래픽 드라이버 버전 최신화

드라이버 버전이 너무 낮으면 CUDA 가속 빌드가 구동되지 않고 에러가 발생할 수 있습니다. [GeForce 드라이버 업데이트]를 진행해 주세요.
### C++ 재배포 가능 패키지 필수

프로그램 실행 시 `DLL load failed` 에러가 발생한다면, PC에 최신 [Microsoft Visual C++ 재배포 가능 패키지](https://aka.ms/vs/17/release/vc_redist.x64.exe)가 설치되어 있는지 확인 후 설치해 주시기 바랍니다.
### 메모리(VRAM) 확보

AI 연산 도중 `Out of Memory` 에러가 발생한다면 실행 중인 고사양 게임이나 비디오 렌더러 프로그램을 잠시 종료한 뒤 재시도해 주세요.

🛠️ FFmpeg 수동 설치 가이드 (자동 다운로드 실패 시)
프로그램 실행 시 FFmpeg 다운로드 단계에서 멈추거나 에러가 발생한다면, 아래 순서대로 직접 다운로드하여 폴더를 배치해 주세요.

1단계: 공식 빌드 파일 다운로드

[Gyan.dev FFmpeg 다운로드 링크](https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.zip)를 클릭하여 압축 파일을 다운로드합니다.

2단계: 압축 해제 및 폴더 이름 변경

다운로드한 ffmpeg-release-essentials.zip 파일의 압축을 풉니다.

압축을 풀면 ffmpeg-7.x.x-essentials_build와 같이 버전 이름이 들어간 폴더가 나옵니다.

이 폴더의 이름을 **ffmpeg**로 변경한 뒤 프로젝트 폴더 내에 넣어주세요. (대소문자 무관, 하이픈이나 버전 숫자 제거)
