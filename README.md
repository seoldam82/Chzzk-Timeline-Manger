# 치지직 타임라인 (Chzzk-Timeline)

치지직(CHZZK) 다시보기(VOD)의 음성 데이터와 실시간 채팅 화력을 분석하여, Gemini AI 기반 하이라이트 타임라인을 자동 생성하고 댓글까지 등록해주는 자동화 도구입니다.

---

# ✨ 프로젝트 개요

이 프로젝트는 방송 전체 흐름 속에서:

* 스트리머 발언
* 시청자 반응
* 화력 집중 구간
* 주요 방송 흐름

을 동시에 분석하여 사람이 직접 정리한 것에 가까운 타임라인을 생성합니다.

핵심 기능:

* Faster-Whisper 기반 음성 전사(STT)
* 채팅 화력 분석 및 노이즈 제거
* Gemini AI 구조화 요약
* 스트리머 발언 기준 싱크 보정
* 치지직 댓글 자동 등록

---

# 🚀 주요 기능

## 🎵 VOD 오디오 및 채팅 수집

* yt-dlp 기반 고속 오디오 다운로드
* ffmpeg 자동 MP3 변환
* 다운로드 캐싱 지원
* 치지직 채팅 전체 로그 저장
* 특정 구간(%)만 선택 분석 가능

---

## 🧠 Faster-Whisper 기반 STT

* CUDA GPU 자동 사용
* CPU fallback 지원
* 긴 방송 자동 슬라이싱 처리
* 한국어 고정밀 음성 전사

---

## 🔥 채팅 화력 분석

다음 요소를 기반으로 하이라이트를 탐지합니다.

* 채팅 밀도
* 폭발 반응 구간
* 반복 반응 빈도
* 감정 반응 집중도

자동 정제 대상:

* 과도한 ㅋㅋㅋ / ㅎㅎㅎ
* 반복 도배
* 특수문자 이모티콘
* 무의미한 스팸 텍스트

---

## 🤖 Gemini AI 기반 타임라인 생성

Gemini AI가 다음 구조로 방송 내용을 정리합니다.

```text
[대주제; 소주제]
[타임스탬프] - 내용
```

예시:

```text
[저스트 채팅; 방송 시작]
[00:03:05] 인사 및 방송 컨디션 이야기

[게임; 랭크 시작]
[01:12:33] 경쟁전 큐 시작 및 팀원 반응
```

---

## 🎯 스트리머 발언 기준 싱크 보정

단순 채팅 폭발 시간이 아니라:

> 스트리머가 실제로 말하기 시작한 시점

기준으로 타임라인 시간을 자동 보정합니다.

이를 통해 실제 다시보기 탐색 정확도를 높입니다.

---

## 💬 댓글 자동 등록

타임라인 생성 후 치지직 댓글로 바로 등록할 수 있습니다.

사용 방식:

```json
"NID_AUT": "",
"NID_SES": ""
```

기본적으로 브라우저(Chrome/Edge)의 로그인 세션을 자동 탐색하며, 실패 시 직접 입력할 수 있습니다.

### 쿠키 추출 방법

1. 네이버 또는 치지직 로그인
2. F12 → Application → Cookies
3. `https://chzzk.naver.com`
4. `NID_AUT`, `NID_SES` 값 복사

# ⚙️ config.json 설정

## 예시

```json
{
    "TARGET_CHANNEL_ID": "치지직_채널_ID",
    "GEMINI_API_KEY": "본인의_API_KEY",
    "NID_AUT": "",
    "NID_SES": "",
    "WHISPER_LANGUAGE": "ko",
    "WHISPER_MODEL": "turbo"
}
```

---

## 주요 항목

| 항목                | 설명              |
| ----------------- | --------------- |
| TARGET_CHANNEL_ID | 분석 대상 치지직 채널 ID |
| GEMINI_API_KEY    | Gemini API 키    |
| NID_AUT / NID_SES | 네이버 로그인 쿠키      |
| WHISPER_LANGUAGE  | Whisper 언어 설정   |
| WHISPER_MODEL     | Whisper 모델 크기   |

---

# 🧠 WHISPER_MODEL 설명

| 모델       | 속도    | 정확도   | 특징       |
| -------- | ----- | ----- | -------- |
| tiny     | 매우 빠름 | 낮음    | 저사양용     |
| base     | 빠름    | 보통    | 기본 추천    |
| small    | 빠름    | 준수    | 밸런스형     |
| medium   | 보통    | 좋음    | 방송 분석 추천 |
| large-v3 | 느림    | 매우 높음 | 최고 정확도   |
| turbo    | 매우 빠름 | 높음    | 최신 고성능   |

추천:

* 빠른 테스트: `base`, `small`
* 고품질 분석: `medium`, `turbo`

# 🤖 모드 1 — AI 타임라인 생성

## 실행 흐름

### 1. 최근 VOD 목록 조회

```text
[0] 방송 제목 A
[1] 방송 제목 B
```

---

### 2. 분석할 영상 선택

```text
분석할 VOD 번호: 1
```

---

### 3. 분석 범위 설정

```text
시작 %: 0
종료 %: 100
```

예시:

* 전체 분석 → `0 ~ 100`
* 후반부만 → `50 ~ 100`

---

### 4. 자동 AI 처리

```text
오디오 다운로드
→ Whisper STT
→ 채팅 분석
→ Gemini 분석
→ 타임라인 생성
```

긴 방송은 1시간 단위 청크로 자동 분할 처리됩니다.

---

### 5. 결과 출력

```text
🤖 AI 자동 생성 타임라인

[저스트 채팅; 방송 시작]
[00:03:05] 방송 인사 및 컨디션 토크

[게임; 랭크 시작]
[01:12:33] 경쟁전 큐 시작
```

---

### 6. 댓글 등록

5000자 이하일 경우 업로드 가능:

```text
댓글 등록하시겠습니까? (y/n)
```

---

### 7. 자동 저장

```text
TL_VOD_[VOD번호]_[시작%]_[종료%].txt
```

예시:

```text
TL_VOD_123456_0_100.txt
```
