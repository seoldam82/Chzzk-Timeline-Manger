# 치지직 타임라인 매니저 (Chzzk-Timeline-Manager)

치지직(CHZZK) 다시보기(VOD)의 오디오 음성 데이터와 실시간 채팅 화력을 종합 분석하여, Gemini AI 기반으로 구조화된 하이라이트 타임라인을 자동 생성하고 치지직 댓글까지 등록해주는 자동화 도구입니다.

---

# ✨ 프로젝트 개요

이 프로젝트는  방송 전체 흐름 속에서:

* 스트리머가 실제로 무엇을 말했는지
* 시청자 반응이 언제 폭발했는지
* 어떤 주제로 방송이 진행되었는지
* 어디가 하이라이트인지

를 동시에 분석하여 사람이 직접 정리한 것에 가까운 형태의 타임라인을 생성합니다.

특히:

* 단순 웃음 도배 제거
* 채팅 화력 압축
* 스트리머 최초 발언 기준 싱크 보정
* AI 구조화 출력
* 댓글 자동 등록

등을 통합한 올인원 치지직 타임라인 자동화 시스템입니다.

---

# 🚀 주요 기능

## 1. 오디오 및 채팅 데이터 수집

### 🎵 VOD 오디오 다운로드

* yt-dlp 기반 고속 다운로드
* ffmpeg 자동 MP3 변환
* 최초 1회 캐싱 후 재사용
* 특정 구간만 다운로드 가능

### 💬 실시간 채팅 데이터 추출

* 치지직 실시간 채팅 로그 전체 수집
* 방송 전체 채팅 로컬 저장
* JSON 기반 후처리 가능

---

## 2. 로컬 AI 기반 음성 전사 (STT)

### 🧠 Faster-Whisper 기반 STT

* CUDA GPU 자동 탐색
* CPU fallback 지원
* 방송 전체를 고정밀 텍스트로 변환

지원 기능:

* 한국어 STT
* 긴 방송 안정 처리
* 자동 슬라이싱
* 메모리 안정성 확보

---

## 3. 채팅 화력 분석 및 압축

### 🔥 채팅 밀도 기반 하이라이트 탐지

다음 요소들을 종합 분석합니다:

* 초당 채팅량
* 특정 시간대 폭발 구간
* 감정 반응 집중도
* 반복 문구 밀집도

### 🧹 텍스트 노이즈 정제

자동 제거 대상:

* 과도한 ㅋㅋㅋㅋ
* ㅎㅎㅎㅎ
* 의미 없는 도배
* 특수문자 이모티콘
* 반복 스팸 문장

---

## 4. Gemini AI 기반 타임라인 구조화

Gemini AI가 단순 요약이 아닌:

* 대주제
* 소주제
* 상황 설명
* 핵심 발언
* 방송 흐름

을 구조화하여 생성합니다.

---

## 5. 스트리머 발언 기준 자동 싱크 보정

단순 채팅 폭발 시간 기준이 아니라:

"스트리머가 실제로 말하기 시작한 시점"

으로 자동 보정합니다.

예시:

❌ 채팅 반응 시작 시점
⭕ 스트리머 발언 시작 시점

이를 통해 실제 다시보기 탐색 정확도가 크게 향상됩니다.

---

## 6. 자동 댓글 등록

다음을 바탕으로 프로그램이 네이버 계정 권한을 빌려 타임라인을 VOD 댓글로 자동 등록합니다.

```config.json
"NID_AUT": "네이버 쿠키에서 추출한 고유 인증 토큰 1 (선택 사항)",
"NID_SES": "네이버 쿠키에서 추출한 세션 인증 토큰 2 (선택 사항)"
```

[상세 설명]
- 역할: 네이버 플랫폼의 본인 인증을 증명하는 실제 로그인 쿠키 값입니다. 프로그램이 타임라인 분석을 마친 후 사용자의 네이버 계정 권한을 빌려 해당 치지직 VOD에 자동으로 댓글을 작성·수정할 때 사용됩니다.
- 작동 및 입력 방식: 기본적으로 공백("")으로 비워두면 프로그램이 PC 내 활성화된 브라우저(Edge, Chrome 등)를 실시간으로 탐색하여 현재 네이버에 로그인된 세션을 자동으로 탈취(우회)하도록 설계되어 있습니다.
- 다만, 브라우저가 완전히 닫혀있거나 권한 문제로 자동 탐색이 실패하는 경우, 또는 특정 네이버 계정으로 댓글 등록을 고정하고 싶을 때만 아래 단계를 통해 직접 입력합니다.

  ※ 쿠키 값 추출 방법:
  1. 웹 브라우저로 네이버 또는 치지직에 로그인합니다.
  2. F12(개발자 도구)를 누른 뒤 [애플리케이션(Application)] -> [쿠키(Cookies)] -> [https://chzzk.naver.com] 메뉴를 차례로 클릭합니다.
  3. 이름 목록에서 'NID_AUT'와 'NID_SES'를 각각 찾아 그 옆에 있는 값(Value)의 긴 문자열을 그대로 복사하여 이곳에 붙여넣습니다.

## WHISPER_MODEL

음성을 분석하여 대본으로 변환할 때 사용할 **Faster-Whisper AI 모델의 크기**를 지정합니다. 

| 모델 크기 (Model) | 연산 속도 | 자막 정확도 | 권장 하드웨어 사양 | 특징 |
| :--- | :---: | :---: | :---: | :--- |
| **`tiny`** | 🚀 매우 빠름 | 📉 낮음 | 저사양 PC / 내장 그래픽 | 속도가 가장 빠르지만, 고유명사나 웅얼거리는 발언의 오차가 심합니다. |
| **`base`** | ⚡ 빠름 | 📊 보통 | 일반 사무용 PC (기본값) | 속도 대비 무난한 성능을 보여주며, 리소스를 적게 먹습니다. |
| **`small`** | 🏃 빠름 | 📈 준수함 | 보급형 그래픽카드 환경 | 적당한 속도와 일상적인 대화 분별력을 갖춘 밸런스형 모델입니다. |
| **`medium`** | ⏰ 보통 | ✨ 좋음 | 메인스트림급 GPU 환경 | 방송용 은어나 발음이 뭉개지는 구간도 비교적 정확하게 잡아냅니다. |
| **`large-v3`** | 🐢 느림 | 🎯 매우 높음 | 고사양 GPU (VRAM 8GB 이상) | 정확도가 가장 뛰어나지만 무겁고 연산 시간이 오래 걸립니다. |
| **`turbo`** | ✈️ 매우 빠름 | 🌟 좋음 | 최신 고사양 GPU 환경 | `large-v3`급의 높은 정확도를 유지하면서 연산 속도를 획기적으로 개선한 최신 모델입니다. |

> 💡 **추천 세팅:**
> * **개인 PC 사양이 평범하거나 빠른 테스트를 원할 때:** `base` 또는 `small`
> * **시간이 조금 걸리더라도 정확한 고품질 타임라인을 원할 때:** `medium` 또는 `turbo` (NVIDIA 외장 그래픽 권장)
